# Data Collection Workflow

Workflow guide for Step 3 of the DAU 9-Step Trade Study Process.

---

## MANDATORY BEHAVIORAL REQUIREMENTS

**This phase CANNOT proceed without:**
1. Approved problem statement from Phase 1-2
2. Complete source registration
3. User confirmation at every decision point
4. Source citation for every data point

**PROHIBITED:**
- Inferring data values from similar products
- Using placeholder or estimated values without explicit user approval
- Proceeding with incomplete data without flagging gaps
- Making any claim without source attribution
