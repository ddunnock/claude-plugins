    1. Create scripts/trace_validator.py with class TraceValidator:
       - DESIGN_ELEMENT_TYPES = {"component", "interface", "contract"} (not needs, requirements, sources, assumptions, traceability-links)
       - TRACE_FIELDS dict mapping slot_type to required trace field names:
         - component: ["requirement_ids"]
         - interface: ["requirement_ids", "source_component", "target_component"]
         - contract: ["component_id", "interface_id", "requirement_ids"]
       - validate(slot_type, content, api) -> list[dict]:
         - Return [] immediately if slot_type not in DESIGN_ELEMENT_TYPES
         - Check each trace field: if missing or empty list, add {"type": "missing_trace_field", "field": field_name, "message": "..."}
         - Check reference existence: for fields that should be slot IDs (requirement_ids items, source_component, target_component, component_id, interface_id), call api.read(ref_id). If None returned, add {"type": "broken_reference", "field": field_name, "ref_id": ref_id, "message": "..."}
         - Return list of warning dicts
       - IMPORTANT: Never raise exceptions. This is warn-but-allow per TRAC-04.

       ```python
       def __init__(self, workspace_root, schemas_dir, journal_path=None, trace_validator=None):

       ```

       - If self._trace_validator is None, return immediately
       - Call self._trace_validator.validate(slot_type, content, self)

         - Log warnings via logger.warning
