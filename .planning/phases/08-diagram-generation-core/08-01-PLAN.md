    - Test: _sanitize_id() replaces hyphens and special chars with underscores
    - Test: generate_d2() renders "unlinked" section slots in a separate container labeled "Unlinked"
    - Test: generate_mermaid() renders "unlinked" section slots as standalone nodes with lighter styling
  </behavior>
  <action>
    Create `scripts/diagram_generator.py` with these pure functions:

    1. `_sanitize_id(slot_id: str) -> str`: Replace non-alphanumeric chars (except underscore) with underscores using `re.sub(r"[^a-zA-Z0-9_]", "_", slot_id)`.

    2. `generate_d2(view: dict) -> str`: Pure function, no side effects.
       - Emit comment header: `# Diagram: {spec_name} (structural)` and `# Generated from snapshot: {snapshot_id}`
       - Iterate `view["sections"]`: components as nested containers `{safe_id}: "{name}" { shape: rectangle }`. For "unlinked" slot_type, wrap in an `Unlinked: "Unlinked" { ... }` container.
       - Non-component slot types (interface, contract, etc.) render as simple labeled nodes.
       - Iterate `view["edges"]`: emit `{src} -> {tgt}: {relationship_type}` connections.
       - Iterate `view["gaps"]`: emit dashed placeholder nodes with `[GAP]` prefix in label, severity-based stroke color (error=#dc3545, warning=#e6a117, info=#888888), `stroke-dash: 5`, `font-color` matching stroke. Add suggestion as `# Suggestion: {suggestion}` comment. Add dashed connection to related context where gap has slot_type matching a section.
       - Handle empty view: just the comment header.

    3. `generate_mermaid(view: dict) -> str`: Pure function, no side effects.
       - Emit comment header: `%% Diagram: {spec_name} (behavioral)` and `%% Generated from snapshot: {snapshot_id}`
       - Emit `graph TD` (default top-down; use `graph LR` if edge count > 2x node count).
       - Iterate sections: emit nodes as `safe_id["Label"]`. Unlinked slots get lighter styling via a `classDef unlinked` with fill:#f8f8f8.
       - Iterate edges: emit `src -->|relationship_type| tgt`.
       - Iterate gaps: define `classDef gapWarning stroke:#e6a117,stroke-width:2px,stroke-dasharray: 5 5,color:#e6a117` (and gapError with #dc3545, gapInfo with #888888). NO commas in stroke-dasharray value. Emit gap nodes with `:::gapClassName`. Add suggestion as `%% Suggestion: ...` comment.
       - Handle empty view: comment header + `graph TD` only.

    4. `_compute_diagram_slot_id(spec_name: str, source: str) -> str`: Returns `diag-{spec_name}-{sha256(source)[:12]}` using hashlib.

    Add corresponding tests to `tests/test_diagram_generator.py` using the `_make_view()` fixture from Task 1. Each test asserts on specific output substrings (not exact match) to verify structural correctness.
  </action>
