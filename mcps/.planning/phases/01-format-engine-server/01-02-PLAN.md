    - XML: parse('<root attr="v">text</root>') returns object with @_attr and #text keys; serialize back produces valid XML; round-trip preserves attributes and text content; optimized for prompt markup (removeNSPrefix:true, parseTagValue:false)
    - TOML: parse('a = 1') returns {a:1}; serialize({a:1}) returns valid TOML; round-trip preserves tables, arrays, strings
    - Registry: getHandler("file.json") returns jsonHandler; getHandler("file.yml") returns yamlHandler; getHandler("file.yaml") returns yamlHandler; getHandler("file.unknown") throws FormatError with code "UNSUPPORTED_FORMAT"; getSupportedExtensions() returns all registered extensions
    - Parse errors: malformed input throws FormatError with descriptive message, line/column when available
  </behavior>
  <action>
1. Create src/formats/json.ts implementing FormatHandler:
   - extensions: [".json"]
   - parse: JSON.parse with try/catch wrapping into FormatError
   - serialize: JSON.stringify(data, null, 2)

2. Create src/formats/yaml.ts implementing FormatHandler:
   - extensions: [".yaml", ".yml"]
   - parse: yaml.load(content) with try/catch wrapping into FormatError. Use default CORE_SCHEMA (not JSON_SCHEMA -- accept standard YAML behavior per research).

   - extensions: [".xml"]
   - Shared options object used by both parser and builder (per research Pitfall 3):

     - textNodeName: "#text"
     - parseTagValue: false (keep values as strings)
     - parseAttributeValue: false
     - removeNSPrefix: true (prompt markup use case -- user decision)
   - parse: new XMLParser(options).parse(content)
   - serialize: new XMLBuilder({ ...options, format: true, indentBy: "  " }).build(data)
   - Import fast-xml-parser

4. Create src/formats/toml.ts implementing FormatHandler:
   - extensions: [".toml"]
   - parse: smol-toml parse(content) with try/catch
   - serialize: smol-toml stringify(data)

   - Export getHandler(filePath: string): FormatHandler -- uses path.extname().toLowerCase() to look up handler; throws FormatError with code "UNSUPPORTED_FORMAT" if not found
   - Export getSupportedExtensions(): string[] -- returns all registered extension keys

     - TOML: tables, array of tables, inline tables

   - getSupportedExtensions returns all extensions
