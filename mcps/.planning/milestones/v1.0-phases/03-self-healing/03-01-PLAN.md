**GREEN phase:** Implement `healData(rawData: Record<string, unknown>, schema: ZodTypeAny): HealResult` in `src/schemas/healer.ts`:

1. Run `schema.safeParse(rawData)`. If success, short-circuit: `{ healed: true, data: rawData, applied: [], remaining: [] }`.
2. Iterate `result.error.issues`. For each issue, determine fix strategy by `issue.code`:
   - `invalid_type` where `received === "undefined"` (missing field): look up schema at path via `getSchemaAtPath()`. If ZodDefault wrapper found, create fix with `fixType: 'default'`, `newValue: _def.defaultValue()`, `confidence: 'safe'`. If no default, `fixType: 'manual'`, `confidence: 'uncertain'`.
   - `invalid_type` where received !== undefined (wrong type): attempt conservative coercion. String->number via `Number()` (reject NaN). String "true"/"false"->boolean (exact match only). Number/boolean->string via `String()`. null/undefined->default if available. If coercion succeeds: `fixType: 'coerce'`, `confidence: 'safe'`. If fails: `fixType: 'manual'`.
   - `invalid_enum_value`: always `fixType: 'manual'`, include allowed `options` in message.
   - All other issue codes: `fixType: 'manual'`.
3. Separate fixes into `applied` (fixType !== 'manual') and `remaining` (fixType === 'manual').
4. Apply `applied` fixes to rawData using `setNestedValue()` helper (walks path segments, creates intermediate objects/arrays as needed).
5. Re-validate with `schema.safeParse(mutatedData)`. Return `{ healed: true/false based on re-validate, data: mutatedData, applied, remaining }`. If re-validate fails, append new issues to remaining.

**Helper functions (internal, not exported):**
- `getSchemaAtPath(rootSchema, path)`: walks ZodObject shape, unwraps ZodDefault/ZodOptional/ZodNullable to find inner types.
- `getDefaultValue(schema)`: checks `_def.typeName === 'ZodDefault'`, calls `_def.defaultValue()`.
- `coerceValue(value, expectedType)`: conservative coercion per locked decisions.
- `setNestedValue(obj, path, value)`: nested setter creating intermediates.
- `getNestedValue(obj, path)`: nested getter for oldValue capture.
