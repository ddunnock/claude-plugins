# Sensitivity Analysis Workflow

Workflow guide for Steps 8-9 of the DAU 9-Step Trade Study Process.

---

## MANDATORY BEHAVIORAL REQUIREMENTS

**This phase CANNOT proceed without:**
1. Approved scoring results from Phase 4-7
2. User selection of sensitivity analysis methods
3. User specification of all analysis parameters
4. User confirmation of sensitivity findings

**PROHIBITED:**
- Auto-selecting sensitivity parameters
- Running all analyses by default
- Interpreting results without user review
- Making recommendations based on sensitivity findings
